    Rules applied in determining which CSS rule would be followed

  • By origin (author, user-agent) and importance (if the declaratory has an !important tag)
    • For CSS3: highest level is the transition declarations
    1. User agent important declarations
    2. User important declarations
    3. Author important declarations
    4. Author normal declarations
    5. User normal declarations
    6. User agent normal declarations
  • By specificity (how specific is the selector)
    1. Inline style - always more specific than non-inline styles
    2. Number of ID (unique) selectors
    3. Number of class selectors, attribute selectors, and pseudo-classes
    4. Number of type selectors and pseudo-elements
    5. Example:
      • p{}
      • p.y{}
      • p#x <- MOST SPECIFIC
  • By order (last one wins)
