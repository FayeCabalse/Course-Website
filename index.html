About Web Systems and Technologies

Course Website

Welcome to the course subject, Web Systems and Technologies. The topics covered in this course are: Internet and the World Wide Web; the HyperText Transfer Protocol (HTTP) and the fundamentals; HyperText Markup Language, Cascading Style Sheets, and JavaScript; Client-side and Server-side Scripting; Web Servers and Web Clients; and the Web Applications. The practical part of this course introduces tools and techniques for specifying and prototyping interactive World Wide Web applications based on HTML, CSS, XML, and many more. The course aims to provide an understanding of the basics of the Internet, how it originated, how it works, and how to develop and maintain Web sites. This course provides the principles and skills of web application development. It arms students with current web programming technology and the skills for developing web oriented applications.

About the Site

This website is created by Information Technology students of Saint Louis University. This site aims to help users to gain knowledge about web developing, server-client interaction of computers using web services, and to understand the different responses that the server sent.
