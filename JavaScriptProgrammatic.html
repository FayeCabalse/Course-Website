JavaScript Values

  • The JavaScript syntax defines two types of values: Fixed values and variable values.
    • Fixed values are called literals.
      • The most important rules for writing fixed values are:
      • Numbers are written with or without decimals.
      • Strings are text, written within double or single quotes.
    • Variable values are called variables
      • Variables are used to store data values.
      • An equal sign is used to assign values to variables.

JavaScript Operators

  • JavaScript uses arithmetic operators ( + - * / ) to compute values.
  • JavaScript uses an assignment operator ( = ) to assign values to variables.

JavaScript Expressions

  • An expression is a combination of values, variables, and operators, which computes to a value.
    •  Example: 2 * 5
  • Expressions can also contain variable values.
    •  Example: a * 5
  • The values can be of various types, such as numbers and strings.
    •  Example: "Webtek" + " " + "Lecture"

JavaScript Keywords

  • Variables with the var keyword becomes a global variable.
      • var a = 10;
function f(){
    var a = 20;
    console.log(a);                     RESULT: 10
    {                                           20
        var a = 30;                             30
        console.log(a);         
    }
    console.log(a);
}
  • Variables with the let keyword only exists within the block.
      • let a = 10;
function f(){
    let a = 20;
    console.log(a);                     RESULT: 20
    {                                           30
        let a = 30;                             20
        console.log(a);         
    }
    console.log(a);
}
  • Variables with the const keyword don't introduct a global property of the variable.
      • const a = 10;
function f(){
    const a = 20;
    console.log(a);                     RESULT: 20
    {                                           30
        const a = 30;                           20
        console.log(a);         
    }
    console.log(a);
}
